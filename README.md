## Quick Start

1. **Clone and install**
   ```bash
   git clone https://github.com/ShozabSHK10/PlayPal.git
   cd PlayPal
   flutter pub get
   ```

2. **Setup Firebase**
   - Create project at [Firebase Console](https://console.firebase.google.com)
   - Add your app

3. **Run**
   ```bash
   flutter run
   ```
